Event
=====

.. class:: event
  .. method:: new(collection , start, end, type, description)
  Construction d'un nouvel évènement.

  :param collection: La collection de l'évènement.
  :type collection: collection
  :param start: date et heure du début de l'évènement.
  :type start: datetime
  :param end: date et heure de la fin de l'évènement.
  :type end: datetime
  :param type: type de l'évènement.
  :type type: string
  :param description: description de l'évènement.
